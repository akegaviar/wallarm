===================================
Поддерживаемые операционные системы
===================================

Wallarm поддерживает установку на следующие операционные системы:

* Debian 7.x (wheezy)
* Debian 8.x (jessie)
* Ubuntu 14.04 LTS (trusty)
* Ubuntu 16.04 LTS (xenial)
* CentOS 6.x
* CentOS 7.x
