===================================
Поддерживаемые операционные системы
===================================

Wallarm поддерживает установку на следующие операционные системы: 

* Debian GNU/Linux 7.x–8.x
* CentOS 6.x–7.x
* Ubuntu 14.04 LTS, Ubuntu 16.04 LTS
