=================================
Обновите модули Wallarm на Linux
=================================

Для обновления модулей Wallarm, установленных на Linux, вам потребуется
выполнить следующие действия:

#. Обновить пакеты Wallarm.
#. Проверьте конфигурационный файл NGINX.
#. Перезапустить NGINX.

Проверьте конфигурационный файл NGINX
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Проверьте, конфигурационный файл NGINX остался корректным после обновления
пакетов.

Выполните команду:
