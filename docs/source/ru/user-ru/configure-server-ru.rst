============================
Настройте Wallarm на сервере
============================

Настройка выполняется на сервере с установленными компонентами Wallarm.

Здесь вы можете:

* Выполнить тонкую настройку в файле :file:`nginx.conf`
* Настроить мониторинг фильтрующего узла, чтобы получать информацию о статусе
  работы.
* Настроить схему резервирования.
* Настроить блокировку по IP.
