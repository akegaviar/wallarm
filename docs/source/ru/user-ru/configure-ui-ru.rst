==================================
Настройте Wallarm в веб-интерфейсе
==================================

Настройка выполняется в `веб-интефейсе Wallarm <https://my.wallarm.com/settings>`_.

Здесь вы можете:

* :guilabel:`Система` -- задать режим работы, режим сканирования, проверку
  векторов атак
* :guilabel:`Узлы` -- посмотреть и удалить узлы
* :guilabel:`Сканер` -- задать режимы работы сканера
* :guilabel:`Приложения` -- задать идентификаторы приложений
* :guilabel:`Маркеры` -- задать маркеры для HTTP-запросов
* :guilabel:`Пользователи` -- создать, активировать и удалить пользователей
